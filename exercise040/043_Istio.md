### Istio

Istio is a service mesh which improves the security inside of the cluster. It enables mtls and many more features.

It is open source and free to use, but like all service meshes has some complexity to it. For testing purposes the Istio documentation offers a broad range of different tutorials

Istio installation
https://istio.io/latest/docs/setup/install/istioctl/
https://istio.io/latest/docs/setup/additional-setup/config-profiles/

Configure traffic shift in Istio
https://istio.io/latest/docs/tasks/traffic-management/traffic-shifting/

For visualisation there is a tool called Kiali, which directly integrates with istio.
https://istio.io/latest/docs/ops/integrations/kiali/
